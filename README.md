# Aspirenet (Prototype)

Gen-Z + Millennial networking app prototype fusing Discord/Tinder socials, LinkedIn-style matching, and an interactive regional map.

## Run

- Prereq: Flutter SDK installed.
- From the project root:


flutter pub get
flutter run -d chrome   # or an emulator/device


## Notes
- Light pastel theme with light-yellow canvas (#FFF9E5).
- Map-centric discovery: pins open event bottom sheets (no separate Events tab).
- Floating Filter on Discover/Map with simple options.
- Feed supports emoji reactions and comments.
- Matching offers two flows: "Find my vibe" (similar) and "Find my balance" (complementary).
- Messaging supports inline polls.
- Profile highlights goals (streaks), skills, achievements (animated badges), and strengths/needs.
